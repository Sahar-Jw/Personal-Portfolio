#  Personal Portfolio

## Overview
A modern, responsive personal portfolio website showcasing my frontend development skills, experience, and projects. Built with React.js, it highlights expertise in scalable web apps, e-commerce, admin dashboards, and more. Includes dark/light theme, interactive project gallery, timeline, contact form, and GitHub integration.

**Purpose**: Present professional work to potential employers/clients, with bio, CV download, and easy contact.

## Features
- Fully responsive (mobile to desktop)
- Dark/Light theme toggle (custom hook)
- Smooth scrolling & scroll-to-top
- Hero intro section
- About bio with skills focus
- Experience/Education timeline
- Featured projects with cards & screenshots
- Contact form + details (phone, email, location)
- React Router for Projects page
- GitHub repo link

## Tech Stack
- **React**: v19.2.4 (with React DOM, Router DOM v6)
- **Build**: Vite v5 (fast HMR)
- **CSS**: Tailwind CSS v4 (via Vite plugin)
- **Icons**: React Icons v5.6.0
- **Linting**: ESLint v9 (React hooks rules)
- **Other**: Custom JSX components, hooks (useTheme), ProjectData.js

## Page Sections (Home.jsx)
1. **Hero**: Personalized intro ("Sahar")
2. **About**: Frontend dev bio
3. **Education/Experience**: Timeline component
4. **Projects**: Cards from data, GitHub link
5. **Contact**: Form + icons/contacts

## Quick Start
```bash
npm install
npm run dev  # http://localhost:5173
npm run build  # Production build
```
